INSTRUCTIONS POUR L'IMAGE DE FOND

1. Pour ajouter une image de fond à votre jeu:
   - Placez une image avec le nom "background.jpg" dans ce dossier (src/resources/)
   - L'image devrait idéalement avoir une résolution minimale de 1024x768 pixels
   - Préférez une image sombre ou à contraste modéré pour ne pas compromettre la lisibilité du texte

2. Si aucune image n'est trouvée:
   - Le jeu utilisera un dégradé de couleurs sombre par défaut
   - Ce dégradé est conçu pour être agréable à l'œil et compatible avec le thème de jeu

3. Format recommandé:
   - JPEG ou PNG
   - Thématique médiévale-fantastique (château, donjon, tour, etc.)
   - Ambiance sombre pour correspondre à l'atmosphère du jeu
